## 构建信息
本项目使用 `yyl` 进行构建

### 安装 yyl
```
npm install yyl -g
```

### 相关命令
```
# 本地开发
npm run d

# history 模式
npm run d:history

# 远程映射
npm run remote

# 打包
npm run all:commit

# 提交代码
npm run commit
```
